Voice Commands

Say these commands by themselves to control the assistant:

Control Commands

  • shutdown - Stop the voice assistant
  • shut down - Stop the voice assistant
  • stop listening - Stop the voice assistant

Speaking Prefix

  • Add a prefix that the AI will say before each response
  • Useful for triggering smart speakers (e.g., "Echo:" or "Alexa:")
  • Leave blank for no prefix

Tips

  • Commands work best when said clearly and alone
  • Use the Settings button to customize the assistant
  • Choose different AI personas for different conversation styles
  • Adjust silence threshold if responses are too fast/slow
